Balises HTML


Introduction de l'HTML

Le HTML est un assemblage de balises qui permettent de créer des pages web.
Une balise est composée de:<nom_de_la_balise attribut1=valeur1 attribut2=valeur2>Texte qui sera dans la balise</nom_de_la_balise>

Le code basique de toutes les pages HTML:
  <!DOCTYPE html>
<html lang="en">
<head>
 <meta charset="UTF-8">
 <title>Titre du document</title>
</head>
<body>
 <!--Mettre les balises ici-->
</body>
</html>

Utiliser le CSS


Introduction au css

Le CSS permet d'ajouter du style à une page web.
Son écriture est composé de:

nom_de_la_balise{
 style1: valeur1;
 style2: valeur2
}

Insérer le css dans une page

Soit la mettre à l'interieur de la page html avec:
<style> Mettre code CSS ici </style>
Soit avec un fichier externe:
<link rel="stylesheet" type="text/css" href="liens/index.css"/>

Ajouter du style à la page

body{
    background-color: black; /*permet de mettre le fond en noir*/
    color: #ffffff /*Et la couleur du texte en blanc en utilisant l'hexadecimal*/
}

Ajouter du style à des balise

h1{
    color: #8B0000; /* Changer la couleur du texte*/
    text-align: center; /* l'alginer au centre*/
    border: 3px /* Et une bordure de 3 pixel*/

}
p{
    color: #ffffff /*Et la couleur du texte en blanc en utilisant l'hexadecimal*/
}

Hover CSS

h1{
    color: #8B0000; /* Changer la couleur du texte*/
    text-align: center; /* l'alginer au centre*/
    border: 3px dashed #505/* Et une bordure de 3 pixel*/                        
}
h1:hover{
    color: #00008B; /* Changer la couleur du texte lors que la souris est dessus*/
    text-align: center; /* l'alginer au centre lors que la souris est dessus*/
    border: 3px solid #055/* Et une bordure de 3 pixel lors que la souris est dessus*/
}

Faire un formulaire


Faire une balise form

Pour créer un formulaire, c'est :
<form method="get ou post" action="file.html"> Données du formulaire </form>
Il existe 2 types de formulaire: les Get et les Post
Les get affiches les information reçu dans l'url et les post non
Et l'action est vers quel fichier renvoie le formulaire une fois compléter.

Ajouter des inputs

<form method="post" action="index.html">
<input name="input"/> <label> Le input permet si on lui met pas de type de renter une chaine de charctères, Quant au label il permet d'afficher du texte à coté du input, le name est la variable qui contient l'information lors de l'envoie</label>
<input name="input" type="password"/> <label> Le type password</label>
<input value="radio" name="input" type="radio"/> <label> Le type radio</label>
<input value="check" name="input" type="checkbox"/> <label> Le type checkbox</label>
<input type="submit"/> <label> Le type subit permet renvoyer ves la page décider précédement dans l'attribut action</label>

</form>

Principe de protocole

Pour le Web, c'est le protocole HTTP(S) (HyperText Transfer Protocole) qui est utilisé.
Il permet de communiquer entre le client(Ex: navigateur web) et serveur(Ex: Data Center)
